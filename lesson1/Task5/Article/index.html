Зміст

  1. Заголовки (h1, h2, h3, h4, h5, h6);
  2. Параграфи (p);
  3. Списки (ul, ol, dl);
  4. Блоки (div);




1.Заголовки

Для визначення заголовків використовуються теги шести рівнів - h1, h2, h3, h4, h5, h6.
Заголовки мають наступний вигляд:

Заголовок 1

Заголовок 2

Заголовок 3

Заголовок 4

Заголовок 5
Заголовок 6





2.Параграфи

Для визначення параграфа вікористовується тег p.
Будь-який текст, заточений между відкриваючим та закриваючим тегами p буде вважатися параграфом.
Тег p являється блочним тегом, тому за замовчуванням буде займати всю ширину сторінки або материнського блока,
висота за замовчуванням буде залежати від контенту.





3.Списки

Для створенн списків використовуються теги ul, ol, dl.
Для визначення элементів списка в середині списків ul и ol використовується тег li.
Для списка типу dl використовуються теги dt - термін і dd - визначення.





4.Блоки

Для створення блока використовують тег div.
Блоки потрібні для заключения сукупності елементів в ньому.
Ширина блока завжди дорівнює ширині сторінки або ширині материнського блока, якщо не задана явно.
Висота блока залежить від змісту, якщо не задана явно.
